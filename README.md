# Suivi.co

## Prerequisites:

  * Running MongoDB instance: `mongod --dbpath /path/to/db`
  * Node.js >= 0.11
  * Grunt (install: `npm install -g grunt`)

## Install

`npm install`

## Run

  * Server: `npm start`
  * Client resources: `grunt ; grunt watch`
